argument-hint: "[CPE-folder-path]"

Please triage the escalation $1 using the Tiger Team Lead approach:

1. Read the folder contents of $1 (README.md, any logs, notes, configs present)
2. Assess the severity (P1/P2/P3/P4) based on what you find
3. Identify which NetApp products are involved and which specialist skills should be loaded
4. Produce a structured triage report covering:
   - **Severity Assessment** with justification
   - **Products & Versions** identified
   - **Problem Statement** (what is actually broken/wrong)
   - **Known Information** (what evidence do we have so far)
   - **Information Gaps** (what do we still need from the customer)
   - **Investigation Plan** (ordered list of investigation steps)
   - **Immediate Actions** (anything that should be done right now)
5. Update `$1/README.md` with the triage findings
6. Load the appropriate specialist skill(s) and begin the investigation
